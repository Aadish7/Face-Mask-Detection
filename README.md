#  Face Mask Detection

##  Objective
Detect whether a person is wearing a face mask using computer vision.

---

##  Approach
- Used a dataset with **with_mask** and **without_mask** images  
- Applied preprocessing and data augmentation  
- Built a model using **TensorFlow/Keras (MobileNetV2 - Transfer Learning)**  
- Used **OpenCV** for face detection  
- Performed binary classification (Mask / No Mask)
